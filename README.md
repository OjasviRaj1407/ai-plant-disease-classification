# AI-Powered Plant Disease Detection using CNN

This project implements a **Convolutional Neural Network (CNN) based deep learning model** to automatically detect plant diseases from leaf images. The model leverages **Transfer Learning and data augmentation** to improve classification performance.

## Project Objective

Plant diseases significantly affect agricultural productivity. This project aims to build an **AI-powered disease detection system** capable of identifying plant diseases from leaf images to support **early diagnosis and precision agriculture**.

## Dataset

The dataset consists of labeled leaf images representing healthy and diseased plants.

Source: PlantVillage Dataset

Classes include:
- Healthy
- Bacterial Spot
- Early Blight
- Late Blight
- Leaf Mold
- Septoria Leaf Spot

## Model Architecture

The CNN model consists of:

- Convolution layers
- ReLU activation
- Max pooling layers
- Dropout layers
- Fully connected dense layers
- Softmax classification layer

Transfer learning techniques were used to improve performance.
